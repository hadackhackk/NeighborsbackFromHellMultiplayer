# NEIGHBORS FROM HELL MULTIPLAYER 
 
## How to Play 
 
1. Run the server first: `python nfh_server.py` 
2. Run clients on other computers: `python nfh_client.py --host SERVER_IP` 
3. Or use the launcher: `python launcher.py` 
 
## Requirements 
- Python 3.8+ 
- PyGame 
- Network connection for multiplayer
